# vfox-apt-larder

[![CI](https://github.com/jbox-web/vfox-apt-larder/workflows/CI/badge.svg)](https://github.com/jbox-web/vfox-apt-larder/actions)

A plugin for the [mise version manager](https://github.com/jdx/mise) to install [apt-larder](https://github.com/jbox-web/apt-larder)

## Install

```
mise plugin install apt-larder https://github.com/jbox-web/vfox-apt-larder.git
```
